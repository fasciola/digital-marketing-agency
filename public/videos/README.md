# Video backgrounds

Add your optimized background videos here.

The website is already prepared to use these files:

- `services-bg.webm` and `services-bg.mp4`
- `strategy-bg.webm` and `strategy-bg.mp4`
- `portfolio-bg.webm` and `portfolio-bg.mp4`
- `results-bg.webm` and `results-bg.mp4`
- `cta-bg.webm` and `cta-bg.mp4`

Recommended settings:

- 5–10 seconds
- muted loop
- WebM + MP4 versions
- compressed for web
- dark, subtle, abstract motion
- 1920×1080 for desktop
- lighter 1080×1920 or 720×1280 versions for mobile if needed

The hero section intentionally does not use video. Video backgrounds are used only after the hero section.
